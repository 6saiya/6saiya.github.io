Monochromatic, 1 pixel per byte array generates an array where each element of the array represents one pixel. It is either on or off.
Monochromatic, 8 pixels per byte, horizontal generates an array where each element of the array represents 8 pixels which lay horizontally.
Monochromatic, 8 pixels per byte, horizontal generates an array where each element of the array represents 8 pixels which lay vertically.
Grayscale generates an array where each element represents one pixel. The value of that element is one of the 16 shades of gray. You can draw whatever color you want it will be converted to grayscale automatically.
Picking color, 24bit per pixel each pixel is represented by three alements in the array. Those are consecutively: red, green and blue components of the pixel.
